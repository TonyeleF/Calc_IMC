# ⚖️ Calculadora de IMC

Aplicação web desenvolvida para calcular o Índice de Massa Corporal (IMC) com base no peso e altura informados pelo usuário, exibindo o resultado e a classificação correspondente.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6)
* Manipulação do DOM

## 📌 Funcionalidades

* 🧮 Cálculo automático do IMC
* 📊 Exibição do resultado na tela
* 🏷️ Classificação do IMC (baixo peso, normal, sobrepeso, obesidade)
* ⚠️ Validação dos campos (peso e altura)
* 🔄 Atualização dinâmica sem recarregar a página
* 📱 Interface simples e responsiva

## 📐 Fórmula do IMC

O cálculo do IMC é feito utilizando a fórmula:

IMC = peso / (altura * altura)

Exemplo:

```id="exemplo1"
IMC = 70 / (1.70 * 1.70) = 24.22
```

## 📊 Classificação do IMC

* Abaixo de 18.5 → Baixo peso
* 18.5 a 24.9 → Peso normal
* 25 a 29.9 → Sobrepeso
* 30 ou mais → Obesidade
